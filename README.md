# Act_1.4 - Wiki - Controls (WinForms)

Aquest repositori conté el projecte de Visual Studio i els recursos visuals utilitzats per a l'activitat **1.4 de la Wiki de Controls**, on s'estudien els controls de Windows Forms.

## 📋 Descripció de l'Activitat
L'objectiu és investigar i documentar el funcionament de diversos controls de Visual Studio. En el meu cas, he seleccionat dos controls de categories diferents per complir amb els requisits de l'activitat.

### Controls Estudiats:
1.  **Button** (Categoria: *Common Controls*)
2.  **SplitContainer** (Categoria: *Containers*)

---

## 📂 Contingut del Repositori

* **/Project:** Conté la solució de Visual Studio `.sln` i el codi font en C# per demostrar el funcionament dels controls.
* **/img:** Carpeta que conté les captures de pantalla utilitzades a la Wiki:
    * `captura-button.png`: Codi d'exemple del control Button.
    * `captura-splitcontainer.png`: Codi d'exemple del control SplitContainer.

## 🛠️ Requisits de l'Entrega (Checklist)

Seguint les instruccions de l'activitat, cada entrada a la Wiki inclou:
- [x] Nom del control.
- [x] URL de la documentació oficial de Microsoft.
- [x] Descripció de la funció del control.
- [x] Propietats més significatives.
- [x] Events més significatius.
- [x] Exemples d'ús.
- [x] Captura de pantalla del codi (allotjada en aquest repo).
- [x] URL del code snippet a GitHub (Gist).
